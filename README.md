## 📂 Projects Completed

### 🌸 1. Iris Flower Classification (Task 1)

- **Objective:** Classify Iris flowers into three species:
  - Setosa  
  - Versicolor  
  - Virginica  

- **Approach:** Built a classification model using Scikit-learn  
- **Outcome:** Achieved high accuracy on test data  
- **Concepts Used:** Supervised Learning, Classification  

---

### 📉 2. Unemployment Analysis with Python (Task 2)

- **Objective:** Analyze unemployment rate trends and study the impact of COVID-19  

- **Process:**
  - Data cleaning  
  - Exploratory Data Analysis (EDA)  
  - Trend visualization  

- **Insights:**
  - COVID-19 significantly increased unemployment rates  
  - Seasonal patterns observed in employment data  

---

### 🚗 3. Car Price Prediction (Task 3)

- **Objective:** Predict car prices using various features such as:
  - Horsepower  
  - Mileage  
  - Brand value  
  - Engine specifications  

- **Approach:**
  - Data preprocessing  
  - Feature engineering  
  - Regression model development  

- **Model Used:** Linear Regression
